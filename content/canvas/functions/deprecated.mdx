---
title: 'Deprecated functions'
description: 'The following Canvas functions are considered deprecated.'
---

It is recommended that you use the alternative functions listed below.

## Canopy elements

<Callout>**Alternatives**: [Canopy Blocks](/docs/canopy/canopy-blocks)</Callout>

The Canopy element functions are deprecated together with the original Canopy editor. Canopy Blocks replace them, letting content editors manage whole sections instead of individual fields. The deprecated functions are:

- [cms_button](/canvas/functions/canopy/cms-button)
- [cms_code](/canvas/functions/canopy/cms-code)
- [cms_heading](/canvas/functions/canopy/cms-heading)
- [cms_image](/canvas/functions/canopy/cms-image)
- [cms_list](/canvas/functions/canopy/cms-list)
- [cms_quote](/canvas/functions/canopy/cms-quote)
- [cms_text](/canvas/functions/canopy/cms-text)

## file

<Callout>**Alternatives**: [include](/canvas/functions/include)</Callout>

**file** returns the file path:

```canvas
{{ file('path/to/file') }}
```

### Arguments

| Argument   | Description          | Type   |
| ---------- | -------------------- | ------ |
| `filepath` | The path to the file | String |

## import

<Callout>**Alternatives**: [include](/canvas/functions/include)</Callout>

**import** returns the contents of a file:

```canvas
{{ import('includes/messages.html') }}
```

<Callout type="warning">The import function no longer renders HTML content within the imported file.</Callout>

### Arguments

| Argument   | Description                                             | Type   |
| ---------- | ------------------------------------------------------- | ------ |
| `filepath` | The path to the file, relative to the `views` directory | String |
